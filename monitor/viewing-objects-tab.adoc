= 檢視「物件」索引標籤
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
「物件」索引標籤提供相關資訊 link:../s3/index.html["S3"] 和 link:../swift/index.html["Swift"] 擷取和擷取速率。

每個儲存節點、每個站台及整個網格都會顯示物件索引標籤。對於儲存節點、「物件」索引標籤也會提供有關中繼資料查詢和背景驗證的物件計數和資訊。
